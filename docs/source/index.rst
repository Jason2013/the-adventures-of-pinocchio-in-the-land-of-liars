《假话国历险记》阅读
====================

《假话国历险记》作者通过笔下的主人公小茉莉的历险，为我们虚构了一个是非颠倒，真假莫辨的国家——假话国。虽然是虚构出来的童话故事，但对于现实中的我们来说，它不仅会带领我们同小茉莉一起进行了一番热热闹闹的历险，更会给我们带来了许多的启示。
